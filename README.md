📌 Ecommerce Sales & Customer Analytics Project


🔹 Business Problem

This project analyzes transactional e-commerce data to uncover revenue trends, customer behavior, and product performance insights.
The goal is to simulate a real-world business intelligence workflow from raw data to dashboard.


🔹 Objectives

> Analyze total revenue and monthly growth

> Identify high-value customers

> Perform RFM segmentation

> Evaluate product performance

> Generate actionable business recommendations


🔹 Tools Used

> Python (Pandas, NumPy, Matplotlib)

> MySQL

> Power BI

> Excel

> Git & GitHub


🔹 Project Workflow

> Data Cleaning (Python)

> SQL Business Analysis

> Advanced Analytics (RFM, Cohort)

> Dashboard Development

> Business Insights & Recommendations


🔹 Expected Insights

> Revenue concentration (Pareto 80/20)

> Customer segmentation behavior

> Seasonal trends

> Product profitability analysis
